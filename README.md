# ZyroMart\n\nA Blinkit-style delivery app with red premium branding, 24-hour delivery windows, and 3 app variants (Customer, Store Owner, Delivery Partner).\n\nBuilt with Flutter, Supabase, and flutter_map.\n\n## Features\n\n- **Customer App**: Browse products, add to cart, place orders, track delivery live on map\n- **Store Owner App**: Manage products, view orders, update order status, analytics dashboard\n- **Delivery Partner App**: Accept deliveries, navigate with map, update delivery status\n- **Red Premium Theme**: ZyroMart branded with `#B71C1C` red theme\n- **24-Hour Delivery**: Extended delivery windows unlike Blinkit's 10-minute model\n- **Live Tracking**: Real-time delivery person location on interactive map\n- **Supabase Integration**: Database, auth, and real-time subscriptions\n\n## Getting Started\n\n```bash\nflutter pub get\nflutter run\n```\n\n## Supabase Setup\n\nRun the SQL schema in `supabase_schema.sql` in your Supabase SQL Editor to create all required tables.\n\nSet environment variables:\n```bash\nflutter run --dart-define=SUPABASE_URL=your_url --dart-define=SUPABASE_ANON_KEY=your_key\n```\n\n## Build APK\n\n```bash\nflutter build apk --release\n```
