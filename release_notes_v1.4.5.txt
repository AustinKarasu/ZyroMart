ZyroMart v1.4.5

Highlights
- Fixed the admin sign-in crash path by guarding all Supabase auth calls behind explicit initialization checks.
- Set production Supabase URL + publishable key defaults for stable release builds.
- Added required store-owner signup fields (store name and store location) and now auto-create/update owner store rows in Supabase after OTP verification.
- Updated order tracking map to always show a red route line from store to customer, with rider marker shown during active delivery.
- Switched order store selection to use live Supabase store catalog data (with fallback), improving map/store linkage.
- Added text normalization in live catalog mapping to prevent corrupted characters in product names and descriptions.

Verification
- flutter analyze
- flutter test
- flutter build apk --release --flavor storefront -t lib/main.dart
- flutter build apk --release --flavor admin -t lib/admin_main.dart
